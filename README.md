# EasyFilters

Easy filters provides the very basic color filters avaliable. 

The color filters used here are:

1. Sepia
2. Grayscale 
3. Hue-rotate 
4. Invert 
5. Saturate
6. Contrast 
7. Brightness 
8. Opacity 
9. Blur 

Code contains the basic HTML page and the CSS file contains all the filter schemes and its usage.
